Pengertian HTML

HTML alias Hypertext Markup Language merupakan bahasa markup yang digunakan untuk membuat halaman website dan aplikasi web. HTML berisi sejumlah kode yang berguna sebagai struktur sebuah website. Kode tersebut berisi kombinasi teks dan simbol yang dibuat berdasarkan format standar kode internasional atau ASCII (American Standard Code for Information Interchange). Anda bisa membuat heading, gambar, memasukkan tautan, dan komponen lainnya dalam halaman website menggunakan HTML. Struktur kodenya pun sederhana, yaitu terdiri dari tag dan atribut sebagai markup pada halaman website.
